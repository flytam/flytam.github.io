大家在日常写博客的过程中，肯定会遇到想迁移博客的时候。例如我之前就遇到想把 csdn 博客文章迁移到 hexo 上。hexo 的文章源文件就是一些符合特定格式 markdown 的文章。因此要实现也比较简单，对 csdn 博客首页 进行爬虫，（都是一些一些基础的获取标签之类的内容操作了，不详细讲了），然后生成符合特定格式 markdown 即可。
